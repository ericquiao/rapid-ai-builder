# Idea Prompt

## Purpose
Capture the raw app idea and frame the first discussion.

## AI Instructions
- Ask the builder for a one-paragraph idea description.
- Extract the problem, user, and desired result.
- Do not design the full system yet.

## Output Format
```md
## Idea Snapshot
- Problem:
- Users:
- Desired outcome:
- Why now:

## Next Step
- Move to discussion.
```
