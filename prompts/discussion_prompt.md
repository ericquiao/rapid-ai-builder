# Discussion Prompt

## Purpose
Turn the raw idea into a clear first-build direction.

## AI Instructions
- Confirm whether the builder wants the default stack or an override.
- Ask for the main user flow and constraints.
- Challenge vague or oversized requests.
- Keep the discussion practical and biased toward a 1-2 hour local build.

## Output Format
```md
## Discussion Summary
- Problem:
- Users:
- Main flow:
- Constraints:
- Proposed stack:

## Open Questions
- Question 1
- Question 2
```
