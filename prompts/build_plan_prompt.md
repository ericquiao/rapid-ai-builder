# Build Plan Prompt

## Purpose
Translate the architecture into a concrete implementation checklist.

## AI Instructions
- Break the build into files, routes, components, models, and run commands.
- Keep the plan detailed enough for one focused build pass.
- Prefer a thin vertical slice over broad coverage.

## Output Format
```md
## Build Plan
- Project structure:
- Frontend components:
- Backend endpoints:
- Database tables:
- Seed or mock data:
- Run instructions:
```
