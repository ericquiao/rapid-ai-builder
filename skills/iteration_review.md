# Name
`iteration_review`

## Purpose
Review the latest iteration and convert raw observations into a small set of decisions, risks, and a clear next action.

## Inputs
- Latest iteration notes
- Builder feedback
- Known issues
- Current project state

## Outputs
- Feedback summary
- Updated decisions
- Updated risks
- Recommended next action

## When To Use
- After builder review
- Before running `/iterate`
- When preparing to update `docs/project_memory.json`
