# /status

## Command
`/status`

## Purpose
Provide a concise dashboard view of the current project state.

## Instructions For AI Execution
1. Read the following project documentation:
   - `docs/project_overview.md`
   - `docs/scope_lock.md`
   - `docs/architecture.md`
   - `docs/build_plan.md`
   - `docs/iterations/`
2. Identify:
   - latest iteration
   - working features
   - known issues
   - next planned improvements
3. Generate a clean summary including:
   - Project Name
   - Goal
   - Current Phase
   - Latest Iteration
   - Working Features
   - Known Issues
   - Next Suggested Iteration
4. If the `docs/iterations/` folder is missing or empty, suggest creating the first iteration record.
