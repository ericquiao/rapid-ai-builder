# /build-app

## Purpose
Generate the application in one focused pass.

## When To Use
Use after the build plan is approved.

## AI Should Produce
- The app code.
- Local run instructions.
- Verification status and known limitations.

## How The AI Should Guide The Builder
- Build only the locked scope.
- Prefer working code over optional abstractions.
- Run the app locally and validate the main flow.
