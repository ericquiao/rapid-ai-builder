# /discuss

## Purpose
Refine the idea into a practical first-build direction.

## When To Use
Use after `/idea` and before requirements or scope lock.

## AI Should Produce
- Discussion summary.
- Clarifying questions.
- Proposed stack choice.

## How The AI Should Guide The Builder
- Ask whether to use the default stack or another stack.
- Identify the main user flow.
- Flag scope that is too large for a 1-2 hour build.
